#+name: Description
#+begin_quote org
Pandoc has this weird built-in table package in latex called =longtable= and it does not support wrapping table. Instead it creates a table with very long lines
#+end_quote

In order to workaround this issue. A filter is need to calculate and fill in the width of each cells within a table. To use the script run:

#+name: Run command
#+begin_src bash
pandoc --lua-filter=table-fix.lua
#+end_src

Or adding to config:

#+name: Pandoc Profile Config
#+begin_src yaml
template: eisvogel

pdf-engine: xelatex

include-in-header:
  - ${USERDATA}/eisvogel-tcolorbox.tex
  - ${USERDATA}/verbatim.tex

filters:
  - table-fix.lua
  - verbatim-style.lua
  - org-html-image.lua
  - org-admonitions.lua
  - mermaid-filter

from: org
to: pdf
#+end_src
